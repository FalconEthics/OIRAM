Welcome to OIRAM, the game that's not a Mario clone! In this wacky adventure, you'll play as OIRAM, a mustached hero on a mission to save the princess from the clutches of an evil mushroom king.

This game was created as a university project using p5.js and vanilla JavaScript. It features charming graphics, catchy soundtrack, and hilarious sound effects that will keep you chuckling as you jump and run through the levels.
